# Support

Thank you for using this project!

If you need help, please use the appropriate channel below so we can assist you efficiently.

## 📖 Documentation

Before opening an issue, please check the project documentation and README. Your question may already be answered there.

## 🐛 Bug Reports

If you've found a bug:

* Search existing issues first.
* If the issue doesn't already exist, open a new Bug Report.
* Include:

  * Operating System
  * Browser (if applicable)
  * Steps to reproduce
  * Expected behavior
  * Actual behavior
  * Screenshots or logs (if available)

## 💡 Feature Requests

Have an idea?

Open a Feature Request issue describing:

* The problem you're trying to solve
* Your proposed solution
* Any alternatives you've considered

## 💬 Questions & Discussions

For general questions, ideas, or community discussions, please use the GitHub Discussions tab instead of opening an issue.

## 🔒 Reporting Security Vulnerabilities

Please **do not** report security vulnerabilities publicly.

Instead, use GitHub Security Advisories or contact the maintainers privately.

## ✅ Before Asking for Help

Please make sure you've:

* Read the README
* Checked the documentation
* Searched existing issues
* Searched discussions
* Updated to the latest version

## ⏱ Response Time

Maintainers review new issues and discussions as time permits. Response times may vary depending on contributor availability.

## 🤝 Code of Conduct

By participating in this project, you agree to follow our Code of Conduct and help maintain a welcoming, respectful community.
